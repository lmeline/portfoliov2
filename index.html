PHP

PHP (Hypertext Preprocessor) est un langage de programmation serveur utilisé principalement pour le développement web

Python

Python est un langage de programmation polyvalent, interprété et facile à apprendre.

C#

C# (C-sharp) est un langage de programmation orienté objet développé par Microsoft.

Javascript

JavaScript est un langage de programmation dynamique principalement utilisé pour le développement web, côté client (navigateur).

My SQL

MySQL est un système de gestion de bases de données relationnelles open-source. Il permet de stocker, organiser et gérer de grandes quantités de données en utilisant le langage SQL (Structured Query Language).

MongoDB

MongoDB est une base de données NoSQL orientée documents, qui stocke les données en format BSON (similaire au JSON), idéale pour les applications nécessitant flexibilité et scalabilité.

Laravel

Laravel est un framework web PHP open-source, conçu pour simplifier le développement d'applications web en fournissant une structure et des outils robustes.

WPF

WPF (Windows Presentation Foundation) est un framework de Microsoft destiné au développement d'applications de bureau Windows.

Html / CSS

HTML (HyperText Markup Language) est le langage standard utilisé pour structurer le contenu des pages web.
CSS (Cascading Style Sheets) est un langage de style qui permet de contrôler l'apparence des pages web créées avec HTML

React

React est une bibliothèque JavaScript développée par Meta, utilisée pour construire des interfaces utilisateur interactives et réactives, principalement côté client, en se basant sur des composants réutilisables.

Node JS

Node.js est un environnement d'exécution JavaScript côté serveur, basé sur le moteur V8 de Chrome, permettant de créer des applications réseau rapides, évolutives et non bloquantes.

Express

Express.js est un framework minimaliste et flexible pour Node.js, conçu pour créer facilement des applications web et des API robustes.
